# sukusyo

Windowsの画面上にある任意の範囲をキャプチャーし、その場に最前面の画像付箋として残せる社内向けツールです。
通常起動するとすぐに範囲選択へ入り、ドラッグを離した瞬間に画像付箋を作成します。

## 主な機能

- ドラッグによる範囲キャプチャー
- クリックによるウィンドウ全体のキャプチャー
- `Ctrl`+クリックによるウィンドウ内容領域のキャプチャー
- キャプチャー画像の最前面表示、複数配置、ドラッグ移動
- クリップボードへのコピー、PNG/JPEG/BMP保存、自動保存
- 蛍光ペン、直線、テキスト挿入、Undo/Redo
- トリミング、縦・横ぶっこ抜き、回転、反転
- クリップボード画像の上下左右への連結
- 50～500%のズーム、透明度変更
- ダブルクリックによる一時退避と自動復帰
- 画像ファイルの読み込み
- ホットキー、タスクトレイ、タスクバーのジャンプリスト
- 単一インスタンス動作。ショートカットをもう一度起動すると新しいキャプチャーを開始

## 使い方

### キャプチャー

`sukusyo.exe`を起動すると、画面が少し暗くなりキャプチャー待ちになります。

| 操作 | 結果 |
|---|---|
| 左ドラッグ | 選択範囲を画像付箋にする |
| 左クリック | クリックしたウィンドウ全体を画像付箋にする |
| `Ctrl`+左クリック | クリックしたウィンドウの内容領域を画像付箋にする |
| 右クリック / `Esc` | キャプチャーをキャンセルする |

クリップボードへ直接コピーするモードは`Ctrl+Shift+A`、画像付箋にするモードは`Ctrl+Shift+P`です。

### 画像付箋

| 操作 | 結果 |
|---|---|
| 左ドラッグ | 画像付箋を移動する |
| `Ctrl`+左ドラッグ | 蛍光ペンで描画する |
| `Ctrl`+`Shift`+左ドラッグ | 直線を描画する |
| `Shift`+左ドラッグ | トリミング・ぶっこ抜き対象を選択する |
| ダブルクリック | 設定時間だけ一時的に隠す |
| 右クリック | 編集・保存・表示メニューを開く |
| `Ctrl+Z` / `Ctrl+Y` | 元に戻す / やり直す |
| `Ctrl+S` / `Ctrl+C` | 保存 / クリップボードへコピー |
| `PageUp` / `PageDown` | 透明度を上げる / 下げる |
| `Esc` | 画像付箋を閉じる |

## 設定

操作パネルまたはタスクトレイの「設定」から、次を変更できます。

- キャプチャー後の自動コピー
- PNG自動保存と保存先
- 画像付箋の最前面表示
- 既定の透明度
- 一時退避時間
- 蛍光ペンの色と太さ

設定は`%LOCALAPPDATA%\sukusyo\settings.json`に保存されます。

## ビルドとテスト

.NET 9 SDKとWindowsが必要です。

```powershell
dotnet build
dotnet run --project tests/Sukusyo.Tests/Sukusyo.Tests.csproj
```

自己完結型の単一実行ファイルを作成する場合:

```powershell
dotnet publish -c Release
```

成果物は`bin/Release/net9.0-windows/win-x64/publish/sukusyo.exe`です。
