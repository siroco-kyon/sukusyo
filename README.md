# sukusyo

Windows のシステムトレイに常駐する、範囲指定スクリーンショットツールです。
選択した範囲を「クリップボードにコピー」するか、「画面上にそのままピン留め」するかを選べます。

## 主な機能

- ドラッグで範囲選択してキャプチャ
- クリップボードへコピー
- 画面にピン留め（常に最前面・ドラッグで移動・複数同時に配置可能）
- ピン留めした画像を保存・コピー・最前面固定のON/OFF
- タスクバーからも操作できる小さな操作パネル

## 使い方

### 起動

`sukusyo.exe` を実行すると、タスクバーに操作パネル（下記参照）が開き、システムトレイにもアイコンが常駐します。
常駐アプリなので、操作パネルを閉じてもアプリ自体は終了しません（バックグラウンドで動き続けます）。

### 範囲キャプチャ（クリップボードへコピー）

1. `Ctrl+Shift+A` を押す（または操作パネル／トレイメニューの「範囲キャプチャ」）
2. 画面が少し暗くなるので、コピーしたい範囲をドラッグして選択
3. マウスを離すと自動的にクリップボードにコピーされる
4. `Esc` でキャンセル

### ピン留めキャプチャ（画面に貼り付けたままにする）

1. `Ctrl+Shift+P` を押す（または操作パネル／トレイメニューの「ピン留めキャプチャ」）
2. 範囲キャプチャと同様にドラッグして範囲を選択
3. 選択した範囲がそのままの位置に、常に最前面表示のウィンドウとして画面に貼り付く

ピン留めされたウィンドウでできること：

| 操作 | 内容 |
|---|---|
| ドラッグ | 好きな位置に移動 |
| 右クリック → 最前面に固定 | 常に最前面に表示するかどうかを切り替え |
| 右クリック → クリップボードにコピー | その画像をクリップボードへコピー |
| 右クリック → 名前を付けて保存 / `Ctrl+S` | PNG・JPEG・Bitmap 形式でファイルに保存 |
| 右クリック → 閉じる / `Esc` | ピン留めを閉じる |

複数回ピン留めキャプチャを行えば、画面上に何枚でも並べて置いておけます。

### 操作パネル（タスクバー）

タスクバーに常時表示される小さいウィンドウから、以下をボタンひとつで実行できます。

- 範囲キャプチャ (`Ctrl+Shift+A`)
- ピン留めキャプチャ (`Ctrl+Shift+P`)
- すべてのピンを最前面へ（他のウィンドウの後ろに隠れたピンを呼び戻す）
- すべてのピンを閉じる
- 終了（アプリを完全に終了する）

ウィンドウ右上の「×」で閉じてもアプリは終了せず、システムトレイに常駐したままになります。
再度開きたいときは、トレイアイコンを右クリック →「ウィンドウを開く」。

### システムトレイ

タスクトレイのアイコンをダブルクリックすると、素早く範囲キャプチャ（クリップボードへコピー）が起動します。
右クリックすると操作パネルと同じ項目に加えて「終了」を選べます。

## ビルド・実行方法

.NET 9 SDK と Windows が必要です。

```
dotnet build
dotnet run
```

配布用に単一の実行ファイルとして発行する場合：

```
dotnet publish -c Release
```

`bin/Release/net9.0-windows/win-x64/publish/sukusyo.exe` に自己完結型（.NET ランタイム同梱）の実行ファイルが出力されます。

## 注意事項

- 同時に起動できるインスタンスは1つだけです（二重起動は無視されます）
- `Ctrl+Shift+A` / `Ctrl+Shift+P` が他のアプリと競合していると、ホットキーの登録に失敗した旨のメッセージが表示されます
